# SlothGame: King of Diamonds

This is a multiplayer game project featuring the **King of Diamonds (Beauty Contest)**. It is built with [Next.js](https://nextjs.org), bootstrapped via [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## 🚀 Features

- Real-time multiplayer gameplay
- Interactive UI with standard animations and responsive design

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org)
- **Database / ORM**: PostgreSQL, Drizzle ORM
- **Styling**: Tailwind CSS
- **Other Tools**: ESLint, TypeScript

## 🏁 Getting Started

First, install the dependencies and run the development server:

```bash
npm install
npm run dev
# or
yarn
yarn dev
# or
pnpm install
pnpm dev
# or
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
